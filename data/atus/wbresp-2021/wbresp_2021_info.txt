
The data file wbresp_2021.dat (2021 ATUS Well-being Module Respondent file) contains 16 variables and 6,902 observations. 

For more information on the 2021 ATUS Well-being Module Respondent file, please see the ATUS Well-being Module Data Dictionary (available at: http://www.bls.gov/tus/wbmintcodebk21.pdf).   

In addition to the data file, three computer programs are contained in wbresp_2021.zip.  If you have SPSS, SAS, or Stata software, you may use these programs to read the data into your statistical software package.  

The programs include variable and value labels for most variables.  For information about unlabeled values, please refer to the data dictionary.


The file "wbresp_2021.sps" reads the data into SPSS; some variable names exceed length 8.

The file "wbresp_2021.sas" reads the data into SAS.

The file "wbresp_2021.do" reads the data into Stata.

If you have SAS, SPSS, or Stata software, open the corresponding program, edit the infile (in SAS), file (in SPSS) or import (in Stata) statement to reference the data file on your computer, and run the program.
